In the 2017 IT Professional New Zealand Code of Ethics, there are 8 Key Codes IT Professionals will practice with:

1. Good Faith

2. Integrity

3. Community-Focus

4. Skills

5. Continuous Development

6. Informed Consent

7. Managed Conflicts of Interest

and 8. Competence.

These 8 codes of the ITP Code of Ethics (ITP meaning IT Professionals) are a part of the Constitute of the Institute of IT Professionals NZ Inc. This ensures all members uphold and follow the guidelines set by the Code of Ethics.
In part 1.5, Section 2 of the IP NZ Code of Ethics, states that " The context of the code is based on 4 key principles that are supported by 4 values (but we will be using only 3).

The principles are:

- Interests of the Community

- Respect for the individual

- Interests of the client

- Professional Integrity

This is supported by the values of:

- Competence

- Truth

- Social Justice
